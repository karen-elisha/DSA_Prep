# Group Anagrams (Java) -- Explanation

## Idea

Two strings are **anagrams** if they contain the same characters in a
different order.

Example:

-   `eat`
-   `tea`
-   `ate`

If we sort each word:

``` text
eat -> aet
tea -> aet
ate -> aet
```

All anagrams produce the same sorted string. We use this sorted string
as the **key** in a `HashMap`.

------------------------------------------------------------------------

## Code

``` java
import java.util.*;

class Solution {
    public List<List<String>> groupAnagrams(String[] strs) {

        HashMap<String, List<String>> map = new HashMap<>();

        for (String word : strs) {

            char[] arr = word.toCharArray();

            Arrays.sort(arr);

            String key = new String(arr);

            if (!map.containsKey(key)) {
                map.put(key, new ArrayList<>());
            }

            map.get(key).add(word);
        }

        return new ArrayList<>(map.values());
    }
}
```

------------------------------------------------------------------------

# Step-by-Step Explanation

## 1. Create a HashMap

``` java
HashMap<String, List<String>> map = new HashMap<>();
```

The map stores:

-   **Key** → Sorted version of the word
-   **Value** → List of words that are anagrams

Example:

``` text
"aet" -> [eat, tea, ate]
"ant" -> [tan, nat]
"abt" -> [bat]
```

------------------------------------------------------------------------

## 2. Traverse every word

``` java
for (String word : strs)
```

This loop visits each string in the input array.

Example:

``` text
eat
tea
tan
ate
nat
bat
```

------------------------------------------------------------------------

## 3. Convert the word into a character array

``` java
char[] arr = word.toCharArray();
```

Example:

``` text
word = "eat"

↓

['e','a','t']
```

------------------------------------------------------------------------

## 4. Sort the characters

``` java
Arrays.sort(arr);
```

After sorting:

``` text
['a','e','t']
```

------------------------------------------------------------------------

## 5. Convert back into a String

``` java
String key = new String(arr);
```

Now:

``` text
key = "aet"
```

Every anagram produces the same key.

------------------------------------------------------------------------

## 6. Check whether the key exists

``` java
if (!map.containsKey(key)) {
    map.put(key, new ArrayList<>());
}
```

If this is the first time we see `"aet"`:

``` text
"aet"

↓

Create new list

↓

[]
```

Store it in the map.

------------------------------------------------------------------------

## 7. Add the original word

``` java
map.get(key).add(word);
```

Example:

First:

``` text
"aet" -> []
```

After adding `"eat"`:

``` text
"aet" -> [eat]
```

After `"tea"`:

``` text
"aet" -> [eat, tea]
```

After `"ate"`:

``` text
"aet" -> [eat, tea, ate]
```

------------------------------------------------------------------------

## 8. Return the answer

``` java
return new ArrayList<>(map.values());
```

`map.values()` returns:

``` text
[
[eat, tea, ate],
[tan, nat],
[bat]
]
```

The constructor converts it into the required `List<List<String>>`.

------------------------------------------------------------------------

# Dry Run

Input:

``` text
["eat","tea","tan","ate","nat","bat"]
```

  Word   Sorted Key   HashMap
  ------ ------------ -------------------------
  eat    aet          aet → \[eat\]
  tea    aet          aet → \[eat, tea\]
  tan    ant          ant → \[tan\]
  ate    aet          aet → \[eat, tea, ate\]
  nat    ant          ant → \[tan, nat\]
  bat    abt          abt → \[bat\]

Final Output:

``` text
[
[eat, tea, ate],
[tan, nat],
[bat]
]
```

------------------------------------------------------------------------

# Complexity

Let:

-   `n` = number of strings
-   `k` = average length of each string

Sorting each word takes **O(k log k)**.

Overall Time Complexity:

``` text
O(n × k log k)
```

Space Complexity:

``` text
O(n × k)
```

------------------------------------------------------------------------

# Key Points to Remember

-   Sort every word.
-   Use the sorted word as the HashMap key.
-   Store all original words with the same key in one list.
-   Return all the lists using `new ArrayList<>(map.values())`.
