# Count Digit Occurrences — Explanation

## Problem
Given an array of integers `nums` and a target `digit` (0–9), count how many times `digit` appears across all the digits of all numbers in the array.

## Code

```java
class Solution {
    public int countDigitOccurrences(int[] nums, int digit) {
        int count = 0;
        int d;
        for (int i = 0; i < nums.length; i++) {
            while (nums[i] != 0) {
                d = nums[i] % 10;
                if (d == digit) {
                    count++;
                }
                nums[i] = nums[i] / 10;
            }
        }
        return count;
    }
}
```

## How It Works

### 1. Outer loop — visit each number
```java
for (int i = 0; i < nums.length; i++)
```
Iterates over every element in the array once.

### 2. Inner loop — peel off digits one at a time
```java
while (nums[i] != 0) {
    d = nums[i] % 10;   // extract last digit
    ...
    nums[i] = nums[i] / 10;  // remove last digit
}
```
This is the classic **"extract last digit, then divide"** trick:
- `nums[i] % 10` gives the rightmost digit.
- `nums[i] / 10` (integer division) chops that digit off.
- The loop repeats until the number becomes `0`, meaning all digits have been processed.

### 3. Compare and count
```java
if (d == digit) {
    count++;
}
```
Every time the extracted digit `d` matches the target `digit`, the counter increases.

### 4. Return the total
After all numbers are fully processed, `count` holds the total occurrences of `digit` across the whole array.

## Walkthrough Example

Suppose `nums = [121, 305, 15]` and `digit = 1`.

| Number | Digits extracted | Matches `1`? |
|--------|-------------------|--------------|
| 121    | 1, 2, 1           | ✅, ❌, ✅ → 2 matches |
| 305    | 5, 0, 3           | ❌, ❌, ❌ → 0 matches |
| 15     | 5, 1              | ❌, ✅ → 1 match |

**Total count = 3**

## Complexity

- **Time:** `O(n × k)`, where `n` is the number of elements and `k` is the average number of digits per number.
- **Space:** `O(1)` — only a few extra variables are used.

## ⚠️ Edge Case to Watch Out For

If any element in `nums` is `0` **and** `digit == 0`, this solution will **miss it**. That's because the `while (nums[i] != 0)` loop never executes when `nums[i]` is already `0` — so the digit `0` inside the number `0` itself is never counted.

**Fix idea:** Add a special check before the while loop:
```java
if (nums[i] == 0 && digit == 0) {
    count++;
}
```

## Side Effect Note

This method **mutates the input array** (`nums[i] = nums[i] / 10` overwrites the original values). If you need to preserve the original array elsewhere in your program, work on a copy instead of the original reference.
